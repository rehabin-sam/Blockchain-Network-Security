# Blockchain Network Security Monitor

A blockchain-powered cybersecurity monitoring dashboard built using:

* Solidity
* Web3.js
* React
* TailwindCSS
* Ganache
* MetaMask

## Features

* Smart contract-based access control
* Real-time blockchain event monitoring
* Immutable audit logs
* Admin authorization/revoke system
* Threat alert feed
* SOC-style cybersecurity dashboard UI

---

## Setup

### Install dependencies

npm install

---

### Start Ganache

Run Ganache on:

http://127.0.0.1:7545

Chain ID:
1337

---

### Deploy smart contract

truffle migrate --reset

---

### Update contract address

Copy deployed contract address into:

src/App.js

---

### Start frontend

npm start
